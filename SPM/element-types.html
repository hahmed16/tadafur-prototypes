SPM Element Types

Define the building blocks your organization uses in strategic plans. Up to 6 types (4 standard + 2 custom).

How element types work: Each type defines what capabilities its elements have (Budget tracking, Progress tracking, Vision alignment). When building a strategy, you choose which types appear and at which levels of the graph. The same type can appear at different levels in different strategies.
Capabilities: Budgetable — tracks planned & actual budget + payments Trackable — has progress log & milestone tracking VisionAligned — can be linked to a Vision Priority Quarterly Measurable — supports yearly baseline with Q1–Q4 target/actual
